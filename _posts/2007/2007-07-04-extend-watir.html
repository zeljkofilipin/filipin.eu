--- tags: code ruby title: Extend Watir ---

I had to click a h4 headline.

{% highlight html %}

Create a new project

{% endhighlight %}

This did not work.

{% highlight ruby %} browser.h4(:id, "addproject").click {% endhighlight %}

I posted a question to wtr-general mailing list and Bret Pettichord suggested to extend Watir with this, so it would support h4 headlines.

{% highlight ruby %} module Watir class H4 < NonControlElement TAG = 'H4' end module Container def h4(how, what) return H4.new(self, how, what) end end end {% endhighlight %}

Now this works.

{% highlight ruby %} browser.h4(:id, "addproject").click {% endhighlight %}
