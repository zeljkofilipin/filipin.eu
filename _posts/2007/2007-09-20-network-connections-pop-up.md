I recently reinstalled Windows XP Professional at my machine. After that, Network Connections pop up started appearing. Yesterday I finally found a way to get rid of it.

Internet Explorer 7 &gt; Tools &gt; Internet Options &gt; Connections &gt; Choose Settings if you need to configure a proxy server for a connection. &gt; Never dial a connection &gt; OK.
