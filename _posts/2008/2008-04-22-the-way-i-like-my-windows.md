I created this reminder the last time I reinstalled Windows XP but it never got out of drafts. Since Windows XP Service Pack 3 will be out soon, I will need this again.

- Download the latest service pack.
- Backup data.
- Make list of applications you use. [Wakoopa](http://wakoopa.com/zeljkofilipin) and [del.icio.us](http://del.icio.us/zeljko.filipin) can help. Do not forget application (mainly Firefox) add ons.
- Install operating system.
- Install service pack.
- Fix Internet connection (Broadcom 440x 10/100 Integrated Controller):  
  Insert hp restore plus! CD &gt; start &gt; My Computer &gt; right click &gt; Manage &gt; Device Manager &gt; Other Devices &gt; Ethernet Controller &gt; right click &gt; Update Driver...
- Activate operating system.
- Install other [updates](http://www.update.microsoft.com/microsoftupdate/v6/default.aspx?ln=en-us).
- Install [antivirus](http://www.grisoft.com/).
- Install [Firefox](http://www.mozilla.com/en-US/firefox/) and [Adobe Flash Player](http://www.adobe.com/shockwave/download/download.cgi?P1_Prod_Version=ShockwaveFlash) plugin.
- Keyboard:  
  start &gt; Control Panel &gt; Date, Time, Language, and Regional Options &gt; Add other languages &gt; Languages - Text services and input languages &gt; Details... &gt; Settings:
  - Installed services &gt; Add.. &gt; Input language &gt; Croatian &gt; OK
  - Default input language &gt; Croatian &gt; Croatian &gt; OK &gt; OK
- Sound (SoundMAX Integrated Digital Audio):  
  Insert hp restore plus! CD &gt; start &gt; My Computer &gt; right click &gt; Manage &gt; Device Manager &gt; Other devices &gt; Multimedia Audio Controller &gt; right click &gt; update driver
- TV:  
  Insert WinFast Drivers (CX2388x) CD &gt; start &gt; My Computer &gt; right click &gt; Manage &gt; Device Manager &gt; Other devices &gt; Multimedia Controller &gt; right click &gt; Update Driver…
  - Multimedia Controller - WinFast TV2000 XP Global/TV Global
  - Multimedia Video Controller - WinFast TV2000 XP Global/TV Global WDM Video Capture
  - Unknown device - WinFast TV2000 XP Global/TV Global WDM Crossbar
  - Unknown device - WinFast TV2000 XP Global/TV Global WDM TVTuner.(Xceive XC2028)
- Monitor (Intel(R) 82845G/GL/GE/PE/GV Graphics Controller):  
  Insert hp restore plus! CD &gt; Start &gt; My Computer &gt; right click &gt; Manage &gt; Device Manager &gt; Other devices &gt; Video Controller (VGA Compatible) &gt; right click &gt; Update Driver…
- Desktop:  
  right click &gt; Properties &gt; Settings &gt; select monitor labeled 1 &gt; check check box Extend my Windows desktop onto this monitor. &gt; OK
- Time:  
  start &gt; Control Panel &gt; Date, Time, Language, and Regional Options &gt; Change the date and time &gt; Time Zone &gt; GMT+01:00 Sarajevo, Skopje, Warsaw, Zagreb
- Regional and Language Options:  
  start &gt; Control Panel &gt; Date, Time, Language, and Regional Options &gt; Regional and Language Options &gt; Regional Options &gt; Standards and formats &gt; Croatian &gt; Location &gt; Croatia &gt; OK
- Windows Explorer:
  - View &gt; Details
  - Tools &gt; Folder Options... &gt; View
    - Folder views &gt; Apply to All Folders &gt; Yes
    - Advanced settings: &gt; Hide extensions for known file types &gt; uncheck &gt; OK
- Remove Internet Explorer clicking sound:  
  Control Panel &gt; Sounds, Speech and Audio devices &gt; Sounds and Audio Devices &gt; Sounds &gt; Program events &gt; Windows Explorer &gt; Start Navigation &gt; Sounds &gt; (None) &gt; OK
