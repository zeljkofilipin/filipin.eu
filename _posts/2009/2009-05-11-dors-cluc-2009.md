Last week I was at [DORS/CLUC 2009](http://www.open.hr/dc2009/). It is three day open source conference in Zagreb, Croatia.

The first day I had a [lightning talk](http://www.open.hr/dc2009/wiki/index.php?page=LightningTalks) titled something like: *How did delivering our web application to client look like before we automated tests and how it looks today?* It was just an introduction to my Watir talk the next day.

The talk was a lot of fun. I really like to show off how Watir is useful, easy to use and, of course, just cool. :) This was the first time I did some live Watir coding using Linux and Firewatir and I had some problems, but nothing serious. (I usually use Windows and Watir::IE.) [Slides](/assets/2009/dors-cluc-2009.pdf) are available.

Also, this was the first time I had a workshop (about 3 hours). We spent a few hours doing some basic Watir tests.

It was nice to see what other people are doing and to meet some familiar faces. There was a lot of intersting presentations, and also a lot of presentations that were a bit boring to me. The usual. The best presentation was by [Dobrica Pavlinušić](http://www.rot13.org/~dpavlin/) on freeing hardware. Topic not really interesting to me, but Dobrica is just the funniest presenter I have seen so far.

There is Twitter [\#dorscluc](http://twitter.com/#search?q=dorscluc) channel, Flickr [dorscluc2009](http://www.flickr.com/photos/tags/dorscluc2009/) tag and \#dorscluc channel at irc.freenode.net projected by one projector all the time.
