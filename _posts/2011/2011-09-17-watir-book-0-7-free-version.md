I have just released Watir Book 0.7 (free version) in PDF, EPUB (iPad, iPhone, iPod) and MOBI (Kindle) format. You can [download the free version of the book](https://github.com/zeljkofilipin/watirbook/downloads) from GitHub for free.

You can [buy the whole book](https://www.paypal.com/cgi-bin/webscr?cmd=_s-xclick&hosted_button_id=WVJATC56MJS3N) via PayPal for 9$. Since the book is not finished yet, you will get all updates to the book for free.

Previous release, 0.6 (July 30) had [498 downloads](https://github.com/zeljkofilipin/watirbook/blob/master/downloads/2011-09-17.png). Release before, 0.5 (July 23) had [55 downloads](https://github.com/zeljkofilipin/watirbook/raw/master/downloads/2011-04-05.png), and 0.4.2 (June 20) had [292 downloads](https://github.com/zeljkofilipin/watirbook/raw/master/downloads/2011-04-05.png). I did not keep record of downloads previous to that. All together, 845 downloads. Not bad.

What is new since the last release?

\- managed to drive all browsers (Internet Explorer, Firefox, Chrome and Opera) on all operating systems (Windows: XP, Vista, 7; Mac 10.5, 10.6; Ubuntu 11.04) (with a few exceptions, see *problems so far* a bit below)  
- fixed images at the [GitHub site](https://github.com/zeljkofilipin/watirbook/)  
- removed Ubuntu 10.10 and Firewatir chapters, since I consider them obsolete  
- using Ruby 1.9.2 on Windows and Ubuntu  
- using build-in Ruby on Mac (1.8.6 on 10.5, 1.8.7 on 10.6)

You can [see all the changes](https://github.com/zeljkofilipin/watirbook/commits/master) at GitHub.

Problems so far:  
- could not drive Firefox 4+ on Mac 10.5, but Firefox 3.6 works just fine  
- could not drive Opera on Windows XP (**update**: [fixed](https://github.com/zeljkofilipin/watirbook/commit/cf1eb409fbb32ef9d690eafe59924e208835bb09)) and Mac 10.5 (**update**: [fixed](http://stackoverflow.com/questions/7483686/can-not-drive-opera-with-watir-webdriver-on-mac-os-x-10-5)) (both fixes will be in 0.7.1, should be released this week)  
- did not cover Mac 10.7 because I do not have it (if three people buys the book this month, I will buy 10.7)

This is the first version of the book that does not include all chapters that I have written so far, but only those that will get you interested in the book enough to buy it. (I hope.) If you do not want to pay, you can [read the whole book](https://github.com/zeljkofilipin/watirbook) at GitHub.

Everybody that already bought the book (all three of them, so far) will get the whole 0.7 version of the book as PDF, EPUB and MOBI files via e-mail.
