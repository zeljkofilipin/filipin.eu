At WebCamp Zagreb 2012 Alesandro Slepcevic mentioned caching plugins for WordPress. I have used WordPress for a long time and I have more than a few WordPress sites, but since all of them are pretty low traffic, I have never had to use caching plugins. After the talk I have decided to install a plugin and see how it works. I have decided to try WP Super Cache on this blog. I am using a couple of free monitoring services (Monitor.Us and Pingdom), so it was just a matter of time to get some data if the plugin helped or not.

The results (for filipin.eu home page) are in. It is pretty obvious that the plugin helped.
