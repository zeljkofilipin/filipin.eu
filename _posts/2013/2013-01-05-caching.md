At [WebCamp Zagreb 2012](http://webcampzg.org/) Alesandro Slepcevic mentioned [caching plugins for WordPress](http://webcampzg.org/alesandro-slepcevic-sto-developeri-rade-da-nama-hosterima-zakompliciraju-zivot/). I have used WordPress for a long time and I have more than a few WordPress sites, but since all of them are pretty low traffic, I have never had to use caching plugins. After the talk I have decided to install a plugin and see how it works. I have decided to try [WP Super Cache](http://wordpress.org/extend/plugins/wp-super-cache/) on this blog. I am using a couple of free monitoring services ([Monitor.Us](http://www.monitor.us/website-monitoring) and [Pingdom](https://www.pingdom.com/)), so it was just a matter of time to get some data if the plugin helped or not.

The results (for filipin.eu home page) are in. It is pretty obvious that the plugin helped.
